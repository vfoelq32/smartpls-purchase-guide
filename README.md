## SmartPLS 4 是什么？为什么研究者都在用它？

SmartPLS 是全球使用最广泛的 PLS-SEM（偏最小二乘结构方程模型）分析软件之一，来自德国，支持 Windows、macOS、Linux 三个平台。

简单说：你有一份问卷数据，想验证几个潜变量之间的路径关系，SmartPLS 就是干这个的。

**它比 AMOS 厉害在哪？**

- AMOS 主打 CB-SEM（协方差结构分析），要求样本量大、数据满足正态分布
- SmartPLS 4 同时支持 PLS-SEM 和 CB-SEM，样本量小（甚至不到 100）照样跑，还对非正态数据更友好
- 可视化建模界面，拖拽就能画路径图，不用写一行代码
- 支持直接导入 Excel/SPSS 格式的数据，不用折腾数据转换

## 功能亮点：SmartPLS 4 相比旧版加了什么？

SmartPLS 4 相比 SmartPLS 3 做了不小的升级：

**算法扩展方面：**
- 新增高斯 Copula 法进行内生性评估（Endogeneity Assessment）
- 新增必要条件分析（NCA，Necessary Condition Analysis）及显著性检验
- 支持回归模型、路径分析与过程建模
- 支持多重调节（三向交互效应）
- 支持标准化、非标准化及均值中心化的 PLS-SEM 分析

**操作体验方面：**
- 内置多种示例模型，新手可以直接上手练习
- 支持报告并排对比（Compare View），切换结果更直观
- 图表高度可定制（颜色、字体、点大小均可调）
- 可根据计算结果直接生成新数据文件，便于二次分析

总之，SmartPLS 4 在保持易用性的同时，把分析能力拉到了一个新高度，更接近科研实战的需求。
